# Real Estate Tech MVP: Virtual Staging Prep & Auto-Tagging

This repository hosts a production-grade MVP for the real estate industry built on **Django**, **OpenCV**, and **Hugging Face Model Pipelines**, with immediate deployment capabilities on **Streamlit Cloud**.

## Key Features
- **OpenCV Enhancement Pipeline:** Real-time perspective warp, edge contour identification, and automated CLAHE luminance adjustment.
- **Deep Learning Feature Auto-Tagging:** Zero-shot object detection to extract property parameters automatically (e.g., detecting chairs, tables, windows, and floors).
- **Hugging Face Dataset Support:** Seamless downloading and local model optimization (Fine-Tuning support via PyTorch).
